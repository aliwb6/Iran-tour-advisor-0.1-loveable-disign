I have a 3-language website (Persian, English, Arabic) and I want to change all fonts 
across the entire site to use a custom font called "Khamenei" font.

The font files are located at:
/home/alireza/Desktop/KHAMENEI Fontes.zip

Please do the following:

1. Extract the zip file at /home/alireza/Desktop/KHAMENEI Fontes.zip
   and check the exact font file names inside

2. Copy all .ttf font files into the project at: public/fonts/ directory

3. Convert or reference them directly as TTF in @font-face 
   (no conversion needed, TTF works in modern browsers)

4. In the global CSS file (src/index.css), add @font-face like this example:
   @font-face {
     font-family: 'Khamenei';
     src: url('/fonts/FILENAME.ttf') format('truetype');
     font-weight: normal;
     font-style: normal;
   }
   (repeat for each weight/style variant found in the zip)

5. Apply the font globally:
   html, body, * {
     font-family: 'Khamenei', sans-serif !important;
   }

6. In tailwind.config.ts or tailwind.config.js add:
   theme: {
     extend: {
       fontFamily: {
         sans: ['Khamenei', 'sans-serif'],
       }
     }
   }

7. Since the site supports Persian, Arabic, and English:
   - Add font-display: swap to @font-face for performance
   - Make sure no other font-family is overriding in any component

8. Search the entire codebase for any hardcoded font-family references 
   and replace them with the Khamenei font

The actual file names must be checked from the zip before writing code.
Use the real filenames found in the zip, do not guess.
